## 📋 What We Collect

PartySpace collects minimal necessary information to:
- Create and manage user accounts
- Facilitate bookings between venues/service providers and clients
- Improve our platform and user experience
- Communicate important updates

## 🔒 Your Data is Protected

We take your privacy seriously:
- Industry-standard security measures
- Data only shared between parties for bookings
- No third-party data sales
- Full user control over account information
